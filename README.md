# Projeto 01: Tabelas Hash em Textos Reais

## Descrição
Este projeto implementa uma Tabela Hash com encadeamento separado do zero e a compara com uma estrutura de busca linear, analisando métricas de espalhamento e colisões usando os textos "Tale of Two Cities" e "Quincas Borba".

## Pré-requisitos
- Python 3.x instalado.
- Os arquivos `tale.txt` e `quincas_borba.txt` devem estar no mesmo diretório do script principal.
---
## Como executar
1. Abra o terminal na pasta do projeto.
2. Execute o comando: `python main.py` (ou o nome do arquivo que contém o código).
3. O programa demorará alguns minutos para rodar (devido à busca linear).
4. Ao finalizar, os resultados serão salvos no arquivo `resultados.csv` no mesmo diretório.
